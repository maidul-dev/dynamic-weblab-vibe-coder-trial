# Performance Guide

## Optimization Checklist

### PHP
- [ ] Minify code where possible
- [ ] Use object caching
- [ ] Avoid direct file calls

### Assets
- [ ] Minify CSS/JS
- [ ] Lazy load JS
- [ ] Use asset versioning

### Database
- [ ] Optimized queries
- [ ] Use transients
- [ ] Index custom tables

## Metrics Targets
- Page load: < 2s
- Widget render: < 100ms
- Memory usage: < 10MB

## Monitoring
- Query Monitor plugin
- Browser dev tools
- Server logs
